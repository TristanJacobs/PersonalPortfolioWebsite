Website Keuzeproject

Tijdens mijn eerste jaar in de ITFactory hadden we een projectweek waar we mochten kiezen in welke richting ons project zich ging afspelen. Ik had samen met mijn vrienden gekozen voor het HTML project. Hiervoor moesten we een website maken voor Actie Min. Dit is een organisatie die eten en hulp biedt aan arme mensen. Ze hadden wel een website maar geen goede.

Onze taak was om hun website op te fleuren naargelang hun eisen. We zijn dus van niks begonnen, het enigste wat we hadden waren de vereisten die door MIN zelf waren opgelegd.

Website

De website is van niets opgebouwd, we zijn van 0 begonnen en hebben zo een zeer uitgebreide website opgesteld waar we redelijk trots op mogen zijn. Er moesten enkele pagina's aanwezig zijn met bepaalde info, we kregen fotomateriaal en reclame afbeeldingen van onze leerkracht die ons ook supergoed geholpen heeft doorheen het project.

Een voorbeeld is dat we een kleine Facebook-feed op de website hebben gezet die altijd up-to-date is.

De website moest vooral gemaakt zijn voor kleinere schermen aangezien het doelpubliek niet veel geld heeft voor grote schermen.

Wat heb ik geleerd?

Tijdens dit project had ik veel geleerd, het gaat van keyboard shortcuts tot de code zelf. Aangezien ik een week bezig was met HTML code heb ik zo superveel geleerd en ervaring opgedaan. Ik heb niet alleen veel van HTML geleerd maar ook van Photoshop omdat we soms een foto moesten aanpassen zodat deze beter op de website zou passen.

Dit was ook de eerste grote teamopdracht dus ik heb daar ook zeker de basis geleerd om in team te werken. Dit is zeer nuttig omdat er nog veel projecten gaan komen.

Besluit

Het keuzeprject was een zeer leuk en leerrijk project waar een zekere basis gelegd wordt die je nodig hebt. Ook was het leuk om eens een echte opdracht te maken met de kennis die we hadden in plaats van kleine websitjes als oefening.
