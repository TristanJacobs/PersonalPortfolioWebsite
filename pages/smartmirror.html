MagicMirror

Tijdens de projectweek van het 1ste semester van het 2de jaar CCS heb ik samen met mijn team een smartmirror gebouwd. Een smartmirror is een display waar een doorkijkspiegel voor staat. Op dit display staan verschillende modules met allemaal andere informatie. Doordat het een doorkijkspiegel is kan je jezelf zien inclusief het display vlak achter het glas.

Dit is allemaal gemaakt op een Raspberry Pi.

Software

Als software hebben we het kant-en-klare pakket van MichMich gebruikt dat op Github staat. In de repository vindt je verschillende modules die je kan invoegen in de basissoftware van MagicMirror.

Alles is super duidelijk uitgelegd en is zeer makkelijk om op te stellen als je een beetje kennis hebt. De installatie is mooi gedocumenteerd.

Er zijn superveel modules die je kan gebruiken. We hebben er een module van het weer ingestoken zodat je real time het weer van je huidige locatie kan bekijken. Ook hebben we er het nieuws ingezet samen met nog andere nuttige modules.

Als extra fun-factor hebben we er een niet zo serieuze pagina in gezet waar je bijvoorbeeld een dagelijkse pok√©mon kan vinden en leuke weetjes over de geschiedenis.

Wat heb ik geleerd?

Het voornamenlijkste dat ik geleerd had is het samenwerken in team met onbekende personen. We hadden al wel af en toe samengezeten maar dit was online wegens de corona-maatregelen dus we kende elkaar niet echt. Maar dit was geen probleem aangezien we goed overeenkwamen en goed samen werkte.

Het omgaan met fouten is een 2de factor dat ik geleerd heb. We hadden redelijk veel foutjes op onze Raspberry Pi en we zijn een paar keer opnieuw moeten beginnen. Gelukkig is alles goed gekomen en hebben we een geod project geleverd.

Besluit

Het was een superleuk project met een leuke ervaring. De software is het grootste deel dat ik echt geleerd heb omdat ik hiervan nog niets wist. Maar het werken in team is zeker ook een kwaliteit die je moet bezitten als je in een groot bedrijf een project moet doen. Met de juiste mensen kan dat een verschil maken van de kwaliteit van het product.
